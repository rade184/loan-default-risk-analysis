# Loan Default Risk Analysis

Excel-based analysis of borrower default risk with practical underwriting recommendations.

## Overview

## Business Problem
Lenders need a simple and reliable way to detect high-risk borrowers before approving loans. This project analyzes applicant characteristics to understand which variables are most strongly linked to default risk.

## Dataset
The dataset includes borrower and loan-related variables such as:
- Credit score
- Debt-to-income ratio (DTI)
- Interest rate
- Loan amount
- Default status

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data binning
- Correlation analysis
- Charts and summary tables
## Skills Demonstrated
- Data cleaning
- Risk segmentation
- Default rate analysis
- Correlation analysis
- Business recommendation writing

- ## Analysis Process
The analysis focused on identifying the variables most associated with default. I grouped continuous variables into meaningful categories, compared default rates across segments, and reviewed correlation results to highlight the strongest risk factors.

## Key Findings
- Credit score showed the strongest relationship with default risk.
- Default rates were much higher in lower credit score groups.
- Borrowers with DTI above 40% showed significantly higher default rates.
- Interest rate was also positively associated with default risk.

## Recommendations
Based on the analysis, the top three risk factors for underwriting are:
1. Credit score
2. Debt-to-income ratio (DTI)
3. Interest rate

Suggested underwriting thresholds:
- Minimum credit score: 600
- Warning threshold for DTI: 40%+
- Higher-interest loans should receive stricter review
